# Frontend Estático — Retratos de Bonito

Versão estática (sem Node/Next.js) do frontend público.

## Arquivos

- `index.html` — Timeline por década
- `search.html` — Busca pública (texto, pessoa, ano, local)
- `photo.html` — Detalhe da foto com histórias e download
- `api.js` — Helpers de fetch para a API

## Servindo

O servidor HTTP já está em execução (porta 3000).

Acesse: **http://127.0.0.1:3000**

## Funcionalidades

- ✅ Timeline histórica por década
- ✅ Busca pública (validadas + públicas apenas)
- ✅ Detalhe da foto (pessoas, histórias)
- ✅ Download com termo de uso
- ✅ Design responsivo (Tailwind via CDN)

## Navegação

- Home (timeline)
- Buscar (formulário avançado)
- Clique em qualquer foto para detalhe

## Design

Usa Tailwind CSS via CDN. Totalmente responsivo (mobile, tablet, desktop).
